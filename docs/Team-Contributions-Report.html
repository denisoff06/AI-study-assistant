StudyAI Team Contributions Report

AI Study Assistant Web Application

University Project | March 2026

1. Project Overview

StudyAI is an AI-powered study assistant that helps university students learn more effectively. Students paste their study notes into the app, and it generates three types of study materials: a structured Summary, interactive Flashcards, and a multiple-choice Quiz. The app is built with a React frontend and a Python Flask backend, powered by the Anthropic Claude API.
